# Comparative Bacterial Genome Analysis

Bioinformatics project analysing genomic composition across three bacterial species:
E. coli, B. subtilis, and S. aureus.

## What this project covers
- Whole-genome GC content calculation
- Nucleotide frequency comparison (A/T/G/C)
- AT and GC strand skew analysis
- Sliding window GC content across genome positions
- ORF (Open Reading Frame) prediction across 6 reading frames

## Tools used
Python · BioPython · Matplotlib · NumPy · NCBI Datasets · Google Colab
